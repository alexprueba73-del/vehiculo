# vehiculo
# Sistema de Control de Vehículo en Python

Este proyecto es un programa desarrollado en **Python** que simula el control básico de un vehículo. Permite acelerar, frenar y verificar si el vehículo está excediendo un límite de velocidad establecido.

## 📌 Descripción

El programa utiliza **Programación Orientada a Objetos (POO)** mediante la clase `Vehiculo`.  
El sistema permite interactuar con el vehículo a través de un **menú en consola** donde el usuario puede realizar diferentes acciones.

Cada vehículo tiene las siguientes características:

- Marca
- Modelo
- Velocidad máxima
- Velocidad actual

El usuario puede modificar la velocidad mediante opciones del menú y verificar si supera un límite de velocidad determinado.

---

## 🧠 Conceptos utilizados

Este proyecto aplica varios conceptos fundamentales de **Python**:

- Clases y objetos
- Método constructor `__init__`
- Métodos de clase
- Condicionales (`if / else`)
- Ciclos `while`
- Menú interactivo en consola
- Entrada de datos con `input()`
